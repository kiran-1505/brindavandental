# Brindavan Dental Care Website

Modern single-page website for Brindavan Dental Care in Banashankari, Bengaluru.

## Stack

- HTML
- CSS
- Vanilla JavaScript
- Ionicons

## Run locally

From the `clinic` directory:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Current site sections

- Hero and appointment CTAs
- Services
- Why patients trust the clinic
- Doctor profiles
- Contact and booking section

## Notes

- The old template assets and placeholder content have been removed.
- The live page entrypoint is `index.html`.
